Installation

installation methoed http://hoojo.cnblogs.com or http://blog.csdn.net/IBM_hoojo. Copy the file, "chatLogs.jar" into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version: 1) go to the plugin screen of the Admin Console, 2) click on the delete icon on the same row as the currently installed search plugin, 3) copy the new chatLogs.jar into the plugins directory of your Openfire installation
